## 🗂 Featured Projects

### 🔍 [Coast Expansion Strategic Analysis](https://github.com/TeresiahNjoroge/coast-expansion-analysis)
**PostgreSQL · DBeaver · SQL**

Go/No-Go analysis for a fresh produce company expanding to a new remote market. Before any analysis could run, I had to find and fix: 85,600+ rows with NULL COGS (reversing the margin conclusion entirely), a NULL filter silently dropping 374 companies, and zero logistics cost records despite the route operating. Final recommendation: Conditional Go, with two pre-launch conditions.

`data quality` `COGS backfill` `synthetic data generation` `business memo`
